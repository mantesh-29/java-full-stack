Introduction to HTML

What is HTML?

  • Full Form:Hyper Text Markup Language
  • Purpose:Standard markup language for creating web pages.
  • Functions:
    • Describes the structure of a web page.
    • Consists of elements (tags) that tell the browser how to display content.
    • Labels content such as headings, paragraphs, links, etc.

HTML Introduction video by Love Babbar

Basic HTML Document Structure

<!DOCTYPE html >
<html>

<head>
<title>Page Title</title>
</head>

<body>
<h1>My First Heading</h1>
<p>My first paragraph.</p>
</body>

</html>

Explaination
< !DOCTYPE HTML > Declares document type as HTML5.
< html > Root element of the page.
< head > Contains meta information (not shown on page).
< title > Title shown in browser tab.
< body > Contains all visible content (headings, text, images, etc.).
< h1 > Large heading.
< p > Paragraph.

HTML Element

Structure:

< tagname > Content are written here </ tagname >

Example:

< h1 > My First Heading </h1>

  • Start tag: < h1 >
  • Content: My First Heading
  • End tag: < /h1 >

Empty Elements: Have no content and no end tag (e.g. , <br>).

Web Browsers

  • Purpose: Read HTML documents and display them according to the tags.
  • Examples: Chrome, Edge, Firefox, Safari.
  • Browsers do not show HTML tags, only the formatted result.

Presentational or Formatting Tags

Presentational (or formatting) tags in HTML are tags whose main purpose is to change the visual appearance of text or content, rather than convey semantic meaning.

These tags were widely used in older HTML versions (HTML 3.2 / 4.01) but are deprecated in HTML5 in favor of CSS for styling. However, browsers still support them for backward compatibility.

Image Tags

In HTML, images are added using the <img> tag, which is self-closing and requires at least the src(source) and alt (alternative text) attributes.

<img> or <img/>

The image can be inserted using src attribute which holds path of the image

<img src= image.jpg alt=Desccription of image />

Form Tag

In HTML , the <form> tag is used to create a section of a web page where the user can enter data and submit it to a server for processing (like login details,search queries, or file uploads).

  • Forms are used to collect the data from web pages.
  • Every form is between 'Form' element.
  • Forms collect the data and send to the server via JavaScript.
  • Form element contains inputs and buttons to submit or reset the forms.

Example: Login from, signup, forget password, servey, feedback, profile, email, newsletter, contact form, appointment form, complaint form.

Basic Syntax

<form action = "server_url" method="get|post"> </form>

Attributes of < form >
