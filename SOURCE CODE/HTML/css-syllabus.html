Introduction to CSS

CSS (Cascadibg Style Sheets)

CSS (Cascading Style Sheets) is a stylesheet language used to control the look and feel (presentation) of a document written in HTML or XML. It separates the content (HTML) from the design (CSS), making websites more attractive and easier to maintain.

CSS Full Form

  • CSS stands for Cascading Style Sheets.
  • Cascading: Refers to the way styles are applied: newer or more specific rules override older or general ones.
  • Style Sheets:A set of rules that define how HTML elements should be displayed.

Purpose of CSS

  • To design and control the layout of multiple web pages at once.
  • To apply styles such as:
    • Colors
    • Fonts & Typography
    • Spacing (margins, padding, line-height)
    • Page Layout (grid, flexbox,positioning)
    • Backgrounds (color, image, gradient),Borders, Animations, Transitions
Advantages of CSS
Advantage Description
Separation of content & design HTML handles content, CSS handles design.
Reusability One CSS file can style multiple pages.
Consistency Uniform look across all web pages.
Faster development No need to repeat style codes inside HTML.
Better performance External CSS reduces page load time by caching.
Responsive Design CSS makes websites adapt to different screen sizes (desktop, tablet, mobile).

CSS can be used in 3 ways in HTML

  • Inline CSS: Styles are applied directly to HTML elements using the style attribute.
  • Internal CSS: Styles are defined within a <style> tag in the head section of the HTML document.
  • External CSS: Styles are defined in separate .css files and linked to the HTML document using the tag.

Inline CSS

  • Definition: Applied directly to an individual HTML element using the style attribute inside the tag.
  • Use case: For quick styling of a single element.
  • Limitation: Not recommended for large projects (difficult to maintain).

Internal CSS

  • Definition: Styles are written within a <style> tag in the head section of the HTML document.
  • Use case: For styling a single HTML Page only.
  • Limitation: Styles are not reusable across multiple pages.

External CSS

  • Definition: Styles are stored in a separate .css file and linked to the HTML file using the <link> tag.
  • Use case: Best for large websites because one CSS file can style multiple pages.
  • Advantage: Makes the website consistent, easy to maintain, and faster.

CSS Selectors

What are CSS Selectors?

CSS selectors are patterns used to select the elements you want to style in your HTML document. They allow you to apply styles to specific elements based on their type, class, id, attributes, and more.

Types of CSS Selectors

  • Element Selector: Selects all elements of a given type (e.g., p, h1).
  • Class Selector: Selects elements with a specific class (e.g., .classname).
  • ID Selector: Selects an element with a specific ID (e.g., #idname).
  • Attribute Selector: Selects elements based on their attributes (e.g., [type="text"]).
  • Pseudo-classes: Selects elements based on their state (e.g., :hover, :first-child).
  • Pseudo-elements: Selects parts of an element (e.g., ::before, ::after).
